# Web scrapping of ISCIII webpage

<https://www.acn.cat/>

## Requisites

`python >= 3.8`

## Usage

```bash
git clone https://github.com/TeMU-BSC/web-scrapping.git
cd web-scrapping/acn/
python3.8 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
pytest test_acn.py
deactivate
```
